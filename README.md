# code-challenge

## Description

This project built by [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm dev

# production mode
$ pnpm run start:prod
```

## Access swagger document

open the watch mode and access `/api/doc` path

```bash
$ pnpm dev
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
